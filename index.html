¿Qué es la programación web?

Es un término que define la creación de sitios web para Internet o una intranet. ... Idealmente un desarrollador web.
debería conocer, además de al menos un lenguaje de programación y el manejo DHTML.
Para conseguirlo se hace uso de tecnologías del lado del servidor y del cliente que conllevan unos procesos de base de datos y el uso de un navegador para realizar determinadas tareas o mostrar información.

Tipos de lenguajes de programación web:

  1. PHP
  2. PERL
  3. JAVA
  4. C++
  5. C#
  6. PHYTON
  7. JAVASCRIPT
  8. RUST... ENTRE OTROS



LENGUAJE EN QUE CONSISTE
PHP Es de código abierto muy usado en el desarrollo web, ya que puede incrustarse en HTML
PERL Está extendido en la construcción de aplicaciones CGI para la web
JAVA Es uno de los lenguajes de programación de código abierto más populares. Java fue comercializado por primera vez 1995
C++ C++ permite la manipulación de objetos y tiene como base el lenguaje C. Se le reconoce como uno de los lenguajes de programación multiparadigma
C# C# se usa para el desarrollo de juegos, servicios web XML, componentes distribuidos, aplicaciones cliente-servidor, etc.
PHYTON Su programación se orienta a objetos y se emplea para hacer aplicaciones genéricas (juegos, web o de escritorio), en la informática científica
